mvvm_news_app
==
[MVVM NewsApp, Retrofit, Room, Coroutines, Navigation Components](https://www.youtube.com/playlist?list=PLQkwcJG4YTCRF8XiCRESq1IFFW8COlxYJ) 튜토리얼에 ListAdapter와 viewBinding을 사용했습니다.

* Coroutines

* Jetpack
